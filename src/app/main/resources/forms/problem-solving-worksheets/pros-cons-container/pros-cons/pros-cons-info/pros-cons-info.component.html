Pros & Cons Example
Problem - I need to learn computer programming.
Possible solution 1 - Join the programming club.
  • Pros - Can get help from others.
  • Cons - Will have to depend on others.
Possible solution 2 - Self-learn from an online course.
  • Pros - Flexibility; can learn any language at any time.
  • Cons - Won’t work on a real project.
